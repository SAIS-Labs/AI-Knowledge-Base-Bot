# 📚 AI Company Knowledge Base Bot

An AI-powered Knowledge Base Assistant built using Streamlit, LangChain, FAISS, HuggingFace Embeddings, and GPT4All.

This application enables organizations to upload multiple company documents and ask questions in natural language. The assistant retrieves relevant information from internal documents and provides accurate answers using Retrieval-Augmented Generation (RAG).

---

## 🚀 Features

* Multiple PDF Upload Support
* Company Knowledge Base Creation
* Semantic Search
* RAG-Based Question Answering
* Chat History
* Source Document References
* Source Page References
* User-Friendly Streamlit Interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/AI-Knowledge-Base-Bot.git
cd AI-Knowledge-Base-Bot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
